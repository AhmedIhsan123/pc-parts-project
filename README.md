## Project Overview

**SurgeForge** is a modern eCommerce web application focused on PC components. The platform is designed to provide users with a streamlined experience for browsing, filtering, and purchasing computer hardware.

The application emphasizes:
- Advanced filtering and search functionality  
- Clean, intuitive UI/UX  
- Scalable backend architecture  
- Structured product data management  

SurgeForge is built to simulate a production-ready online retail environment, demonstrating full-stack development skills including RESTful API design, data modeling, and frontend integration.

---

## Product Category

SurgeForge specializes in **PC hardware components**, including:

- Graphics Cards (GPUs)
- Central Processing Units (CPUs)
- Motherboards
- RAM (Memory)
- Storage (SSD / HDD)
- Power Supply Units (PSUs)
- PC Cases
- Cooling Systems
